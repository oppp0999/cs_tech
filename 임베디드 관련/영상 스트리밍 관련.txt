nginx-rtmp -> 오픈소스 웹 서버인 nginx를 이용한 라이브 스트리밍 서버.
RTMP(real-time-messaging-protocol) 를 지원 -> 가볍고 설치가 간편

OpenCV는 스트리밍을 처리하기 위한 라이브러리는 아님 -> 그래서 미디어 서버와 함께 사용


실시간 -> 데이터가 생성되는 즉시 즉각적으로 처리되는 것을 의미
스트리밍 -> 대량의 데이터를 작은 단위로 나누어 전송하면서 실시간으로 수신측에서 처리하도록 하는 방식


2. 서버/백엔드에서 영상 처리를 할 수 있는 관련 알고리즘
2.1 객체 탐지 -> 이미지나 영상에서 특정 객체를 인식하고 탐지하는 기술 -> 대표적인 알고리즘 YOLO,SSD 등
2.2 얼굴 인식 -> 영상에서 얼굴을 인식하고 인식된 얼굴을 기반으로 신원 확인
2.3 이미지 분할 -> 이미지를 픽셀 단위로 분할하여 각각의 픽셀이 어떤 객체에 속하는지 판별하는 기술
2.4 영상 압축 -> 영상 데이터를 압축하여 전송량을 줄이는 기술
2.5 모션 추적 -> 영상에서 움직이는 객체를 추적하고 분석하는 기술
2.6 실시간 영상처리 -> 영상 데이터를 실시간으로 처리하며, 대표적인 알고리즘 opencv 등
